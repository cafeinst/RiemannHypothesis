# RiemannHypothesis

This repository contains an Isabelle/HOL formalization of a conditional
metamathematical barrier theorem for the Riemann Hypothesis.

The development does **not** claim that the Riemann Hypothesis is
unprovable in ZFC, Peano Arithmetic, or any other standard foundational
system. Instead, it formalizes an abstract proof model whose assumptions
are stated explicitly and proves that the Riemann Hypothesis is not
provable within that model.

This formalization develops and refines the central metamathematical ideas
presented in:

Craig A. Feinstein, *The Riemann Hypothesis is Unprovable*,
arXiv:math/0309367.
